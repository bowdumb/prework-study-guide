Prework Study Guide

HTML

  • Semantic elements are used to indicate the role of their contents, as well as provide structure to the webpage
  • The HEAD element contains information about the webpage for the browser
  • The HEADER elements must be ordered from 1-6
  • The BODY element represents the visible content shown to the user
  • The element stands for "anchor" and creates links to the same webpage or other webpages
  • The tag is used to identify elements for purposed such as CSS styling

CSS

  • CSS stands for Cascading Style Sheets
  • CSS is used to style the webpage and is used alongside HTML and JavaScript which provide the foundations.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside and element.
  • Add a "." before assigning a CLASS element as the selector to apply those CSS property assignments to all of that CLASS.
  • There are three types of CSS implementation: Inline, Internal, and External.

Git

  • Git and GitHub provide version control for software projects
  • Git allows you to create separate brances that can be worked on independently
  • GitHub can be used to create, access, and store repositories
  • git status: Checks what branch we are currently on
  • git checkout -b branch-name: Creates a new branch and switches to it

JavaScript

  • JavaScript is used to integrate logic and functionality into webpages
  • JavaScript powers many of the electronic elements of our modern lives!
  • We will learn more about JavaScript in upcoming lessons
